# AI に Mockery モックを作らせるための仕様

## 目的

* テストコードの作成を自動化する
* Interface に基づいたモックを生成する
* Request バリデーションや戻り値を自動で作成する
* AI が推測で書かないようにルールを明確化する

## AI が作るべき内容

1. 対象 Service Interface を受け取る
2. 呼び出すメソッド名と引数を OpenAPI / Request 仕様から参照
3. 戻り値の型やサンプルデータを OpenAPI / Resource 仕様から参照
4. Request の Validator ルール確認

## 出力フォーマット

* PHP のテストクラス
* Mockery の shouldReceive()->with()->andReturn() が含まれる
* tearDown で Mockery::close() を呼ぶ
* Request の Validator チェックを含む

## 注意事項

* 実際の DB にはアクセスしない
* 他の Service はすべてモックに置き換える
* OpenAPI / Resource の仕様に基づき戻り値を作る
