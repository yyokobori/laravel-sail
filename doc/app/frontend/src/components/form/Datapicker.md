# DatePicker コンポーネント仕様

## 目的

* 再利用可能な日付選択コンポーネントを提供する
* 表示形式・ロケール・スタイルを柔軟に変更可能にする
* UIと日付ロジックを分離し、保守性を高める

---

## 基本仕様

* 入力欄とカレンダーUIで構成する
* 入力欄右端にカレンダーアイコン（SVG）を表示する
* 入力欄またはアイコンをクリックするとカレンダーを表示する
* カレンダーは入力欄の下にポップアップ表示する
* カレンダー外クリックで閉じる

---

## 状態管理

* 内部では Date オブジェクトを使用する
* タイムゾーンはローカル前提とする
* value が null の場合は未選択状態とする（入力欄は空表示）
* placeholder を表示可能とする（例：YYYY/MM/DD）
* カレンダー初回表示時は今日の日付を選択状態とする

---

## 表示形式

* 日付表示形式はフォーマット指定可能とする
* 例：YYYY/MM/DD(曜日)

---

## ロケール

* 曜日は以下の切り替えに対応する

  * 日本語（例：月）
  * 英語短縮（例：Mon）

---

## 曜日表示

* 曜日は太字で表示する
* 日曜始まり / 月曜始まりを props で切り替え可能とする

---

## 曜日カラー

* 日曜日は赤
* 土曜日は青
* 上記カラーは props で変更可能とする

---

## カレンダー表示

* 年月に対応するカレンダーを表示する
* 今日の日付をハイライト表示する（文字を太字にする）
* 選択中の日付を強調表示する（背景色などで区別）

---

## 日付選択動作

* 日付クリック時の動作

  * 選択日を入力欄に反映
  * カレンダーを閉じる

---

## 年・月選択

### 年選択

* スロット形式（縦スクロール）で表示する
* 選択可能範囲内の年リストを表示する
* クリックで年を確定する
* 確定後、該当年月のカレンダーを表示する

### 月選択

* スロット形式（縦スクロール）で表示する
* 1〜12月を表示する
* クリックで月を確定する
* 確定後、該当年月のカレンダーを表示する

---

## フッター操作

* カレンダー下部中央に「リセット」リンクを表示する

  * 押下時の動作

    * 入力欄を未選択状態にする
    * カレンダーを閉じる

* カレンダー下部右に「今日」リンクを表示する

  * 押下時の動作

    * 今日の日付を入力欄に設定する
    * カレンダーを閉じる

---

## 日付制限

* 選択可能な日付範囲を指定可能とする

  * minDate
  * maxDate
* 範囲外の日付は選択不可とする

---

## props仕様

* value: Date | null
* onChange: (date: Date | null) => void
* locale: 'ja' | 'en'
* startOfWeek: 'sun' | 'mon'
* weekendColors:

  * sunday: string
  * saturday: string
* format: string
* placeholder: string
* minDate?: Date
* maxDate?: Date
* styleClasses:

  * container
  * input
  * icon
  * calendar
  * header
  * day
  * selectedDay
  * today
  * footer

---

## スタイル仕様

* Tailwind CSS を使用する
* px指定は禁止（remベース）
* styleClasses により全要素のスタイルを上書き可能とする
* デフォルトスタイルは最小限とする

---

## 日付処理
* 日付処理はコンポーネント内に書かない
* すべて関数として外部に切り出す
* 副作用を持たない純粋関数として実装する
* app/utils/date.ts を使用する

---

## 禁止事項

* 外部日付ライブラリの使用禁止
* CSSファイルの追加禁止
* インラインスタイル禁止
* コンポーネント内で日付計算を直接行うことを禁止（utilsへ分離する）
* 日付処理はコンポーネント内に書かない
