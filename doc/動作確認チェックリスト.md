## 🔗 API連携確認（バックエンド↔フロントエンド）

### 1. Laravelにテストエンドポイントを作成
```bash
docker compose exec backend php artisan make:controller TestController
```

`app/backend/app/Http/Controllers/TestController.php` を編集：
```php
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class TestController extends Controller
{
    public function test()
    {
        return response()->json([
            'message' => 'Hello from Laravel!',
            'timestamp' => now()->toDateTimeString()
        ]);
    }
}
```

`app/backend/routes/web.php` または `routes/api.php` にルート追加：
```php
// routes/api.php
Route::get('/test', [TestController::class, 'test']);
```

### 2. フロントエンドから呼び出し
`app/frontend/src/App.tsx` を編集：
```tsx
import { useState, useEffect } from 'react'
import axios from 'axios'
import './App.css'

function App() {
  const [count, setCount] = useState(0)
  const [apiData, setApiData] = useState<any>(null)

  useEffect(() => {
    axios.get('/api/test')
      .then(response => {
        setApiData(response.data)
        console.log('API Response:', response.data)
      })
      .catch(error => {
        console.error('API Error:', error)
      })
  }, [])

  return (
    <>
      <div>
        <h1>Laravel + React</h1>
      </div>
      <div className="card">
        <button onClick={() => setCount((count) => count + 1)}>
          count is {count}
        </button>
        {apiData && (
          <div>
            <p>API Message: {apiData.message}</p>
            <p>Timestamp: {apiData.timestamp}</p>
          </div>
        )}
      </div>
    </>
  )
}

export default App
```

### チェック項目
- [ ] ブラウザのコンソールに "API Response: ..." が表示される
- [ ] 画面に "Hello from Laravel!" が表示される
- [ ] ネットワークタブで `/api/test` のリクエストが成功（200 OK）
