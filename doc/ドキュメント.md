

# ドキュメント・UI・API自動生成プロジェクト

このプロジェクトは、React（UI）、TypeScript（コード）、Laravel（API）の各種自動生成ドキュメントを統合管理します。

- UIドキュメント: Storybook
- コードドキュメント: TypeDoc
- APIドキュメント: Scribe
- ポータル: Docusaurus（docs-site）

---

## 主要コマンド

### Storybook
- 開発サーバー起動: `npm run storybook`（app/frontend）
- 静的ビルド: `npm run build-storybook`（app/frontend）

### TypeDoc
- ドキュメント生成: `npx typedoc`（app/frontend）

### Scribe
- ドキュメント生成: `php artisan scribe:generate`（app/backend）

### docs-site
- 開発サーバー起動: `npm start`（app/frontend/docs-site）
- 静的ビルド: `npm run build`（app/frontend/docs-site）

---

## ディレクトリ構成

- app/frontend/.storybook/ ... Storybook設定
- app/frontend/src/ ... React/TypeScriptソース
- app/frontend/docs/code/ ... TypeDoc出力
- app/backend/config/scribe.php ... Scribe設定
- app/backend/public/docs/ ... Scribe出力
- app/frontend/docs-site/ ... Docusaurus

## CI/CD自動化例（GitHub Actions）

.github/workflows/docs.yml
```yaml
name: Build Docs
on:
  push:
    branches: [main]
jobs:
  build-docs:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v4
        with:
          node-version: '18'
      - name: Install frontend dependencies
        run: |
          cd app/frontend
          npm ci
      - name: Build TypeDoc
        run: |
          cd app/frontend
          npx typedoc
      - name: Build Storybook
        run: |
          cd app/frontend
          npm run build-storybook
      - name: Install backend dependencies
        run: |
          cd app/backend
          composer install --no-interaction --prefer-dist
      - name: Build Scribe
        run: |
          cd app/backend
          php artisan scribe:generate
      - name: Build docs-site
        run: |
          cd app/frontend/docs-site
          npm ci
          npm run build
```
